# EkaPlace
EkaPlace est une plateforme web réalisée par Marcellin, Harmonie, Micheline, Medi et Gabriel, 5 étudiants en Sciences Informatiques de l’UCB dans le cadre du projet tutoré de fin de la première année universitaire.  Elle met en relation les propriétaires de maisons à louer à Bukavu ainsi que les locataires à la recherche d’une maison d'habitation.
